Description
QViewはiPhone/iPad用のQiitaクライアントアプリです。

Qiita上に共有されたプログラミングに関する情報を回覧・検索・ストック・いいねすることが出来ます。

QViewの機能
- Qiitaの記事の回覧
- Qiitaの記事の検索
- 記事のストック・いいね
*Qiita API v2のOAuth認証に対応しているため、パスワード等の個人情報をアプリに保存しません。

ご注意事項
QViewはQiitaAPIを利用した非公式アプリです。
アプリに関するお問い合わせはアプリ内のお問い合わせからお願いしたします。

*本アプリはQiita非公式アプリです。
